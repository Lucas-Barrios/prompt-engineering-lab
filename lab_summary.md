# Lab Summary

I systematically diagnosed and fixed three failing chatbot prompts through empirical testing. v1 prompts (zero-shot, no structure) showed 10-30% consistency across 15 runs due to uncontrolled output formats and missing constraints. v2 added explicit format requirements and role definitions, jumping sentiment and extraction to 100% consistency but leaving product descriptions variable. v3 introduced few-shot examples and temperature=0, achieving 95-100% consistency across all tasks. The techniques that drove improvement were: constraint injection (ONLY, exact), schema specification for JSON outputs, few-shot examples to anchor style, and temperature=0 for determinism. If doing this again, I'd start with schema definitions earlier and run fewer iterations on v1 to save time—the failure patterns were clear after 5 runs.
